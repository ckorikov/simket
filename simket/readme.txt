для установки mkdocs в командной строке задаётся команда - 'pip install mkdocs'
для установки темы 'material' используется команда 'pip install mkdocs-material'

'mkdocs.yml' - файл конфигурации
папка 'docs' - папка с вкладками сайта 
папка 'img' содержит в себе favicon - иконку, которая будет отображаться вокне браузера
'mkdocs serve' - команда, используется для запуска встроенного dev-server, 
который позволяет в реальном времени просматривать изменения, 
которые создаются в файле конфигурации mkdocs.yml и других файлах
'mkdocs build' - команда, для сборки сайта на основе файлов конфигурации


To install mkdocs, in the command line use command - 'pip install mkdocs'
To install the 'material' theme use the command 'pip install mkdocs-material'
'mkdocs.yml' - configuration file
'docs' folder - site tabbed folder
'img' folder contains favicon - an icon that will be displayed in the browser window
'mkdocs serve' is a command used to run the built-in dev-server,
which allows you to view changes in real time,
which are created in the mkdocs.yml configuration file and other files
'mkdocs build' - command to build a site based on configuration files
